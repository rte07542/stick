내려받기
git status //멀쩡한지 확인 하기 (nothing to commit, working tree clean)
git pull //폴더 있을때
git clone //폴더 없을떄


업데이트 하기
git status //멀쩡한지 확인 하기 (nothing to commit, working tree clean)
git add .
git commit -m "커밋 메시지"
git push origin main


업데이트 할때 status가 오류 났을떈?
git rm -r --cached .   //git이 추적하고 있는 모든 파일의 캐시를 삭제
git add .
git commit -m "커밋 메시지"
git push
